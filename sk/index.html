Rufus

Vytvorte bootovací USB kľúč rýchlo a jednoducho

Rufus je utilita, ktorá pomôže naformátovať a vytvoriť bootovacie jednotky USB Flash, ako sú napr. USB kľúče, pamäťové karty, atď.

Môže to byť užitočné v prípadoch, keď:

  • potrebujete vytvoriť inštalačné USB zariadenie z bootovacieho ISO súboru (Windows, Linux, UEFI, atď.)
  • potrebujete pracovať s počítačom, na ktorom nie je operačný systém
  • potrebujete flashovať BIOS alebo iný firmware cez DOS
  • chcete spúšťať low-level utilitu

Aj napriek kompaktnej veľkosti, Rufus má všetko, čo potrebujete

Navyše, Rufus je rýchly. Napríklad, je dvakrát rýchlejší ako UNetbootin, Universal USB Installer alebo Windows 7 USB download tool pri vytváraní inštalačného USB zariadenia z ISO obrazu Windows 7. Je oveľa rýchlejši aj pri vytváraní bootovacích USB zariadení Linuxových distribúcií z ISO obrazov. (1)
Stručný zoznam podporovaných ISO obrazov sa nachádza nižšie na tejto stránke. (2)

Využijem túto príležitosť pre vyjadrenie vďačnosti prekladateľom, ktorí umožnili programu Rufus a tejto web stránke, aby bola preložená do mnohých jazykov. Ak ste radi, že používate Rufus vo svojom rodnom jazyku, mali by ste sa im poďakovať!

Návod na použitie

Stiahnite súbor EXE a spustite ho – nevyžaduje žiadnu inštaláciu.

Spustiteľný EXE súbor je digitálne podpísaný a tento podpis by mal obsahovať:

  • "Akeo Consulting" (v1.3.0 alebo nižšia)
  • "Pete Batard - Open Source Developer" (v1.2.0 alebo nižšia)

Poznámky k podpore DOS:

Ak vytvoríte bootovaciu jednotku a používate iné ako americké (US) rozloženie klávesnice, Rufus sa pokúsi vybrať správne rozloženie podľa regionálnych nastavení systému. V takom prípade, FreeDOS, ktorý je predvolená možnosť, je odporúčaný namiesto MS-DOS, pretože podporuje viac rozložení klávesnice.

Poznámky k podpore ISO obrazov:

Všetky verzie Rufus od v1.1.0 umožňujú vytváranie bootovacích USB zariadení z ISO obrazu (.iso)

Vytvorenie ISO obrazu z fyzických diskov alebo zo skupiny súborov je veľmi ľahké použitím aplikácie na napaľovanie CD, ako napríklad voľne dostupné InfraRecorder alebo CDBurnerXP.
