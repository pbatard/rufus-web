Rufus

สร้างไดรฟ์บูตUSBอย่างง่ายดาย

Rufusคือโปรแกรมที่ช่วยฟอร์แมทและสร้างUSBแฟลชไดรฟ์ที่สามารถบูทได้ เช่นคีย์ USB / pendrives, Memory Sticks, ฯลฯ

Rufus อาจจะเป็นประโยชน์อย่างยิ่งในกรณีที่:

  • คุณจำเป็นต้องสร้างสื่อการติดตั้ง USB จาก ISOs ที่สามารถบูตได้ (Windows, Linux, UEFI ฯลฯ )
  • คุณจำเป็นต้องทำงานในระบบที่ไม่ได้มีการติดตั้งระบบปฏิบัติการไว้ก่อนหน้า
  • คุณจำเป็นต้องแฟลชไบออสหรือเฟิร์มแวร์อื่น ๆ จาก DOS
  • คุณต้องการเรียกใช้ยูทิลิตี้ระดับต่ำ

แม้จะมีขนาดที่เล็กแต่Rufusมีทุกสิ่งที่คุณต้องการ!

มากไปกว่านั้น Rufusยังสามารถทำงานได้ รวดเร็ว ยกตัวอย่างเช่น Rufusมีความเร็วเกือบจะเป็นสองเท่าของ UNetbootin , Universal USB Installer หรือ Windows 7 USB download tool ในการสร้างไดรฟ์ USB ติดตั้งของ Windows 7 จากISO นอกจากนี้Rufusยังเร็วกว่าเล็กน้อยในการสร้างระบบปฏิบัติการLinuxที่สามารถบูทได้จากUSB จากISOs (1)
รายการ ISOs ที่Rufusสนับสนุนอย่างคร่าวๆ สามารถดูได้จากด้านล่างของหน้านี้ (2)

ข้อกำหนดขั้นต่ำของระบบ:

Windows 7 32หรือ 64บิท หรือระบบปฏิบัติการที่ใหม่กว่า เมื่อติดตั้งแล้วโปรแกรมจะสามารถใช้งานได้ทันที

ผมขอใช้โอกาสนี้เพื่อเป็นการแสดงความขอบคุณอย่างสุดซึ้งไปยังผู้แปลทุกคนที่ทำให้Rufusมายังจุดนี้ได้ และรวมถึงผู้แปลเว็บไซต์ในหลากหลายภาษา หากคุณพบว่าคุณสามารถใช้Rufusในภาษาของคุณเอง คุณควรจะขอบคุณพวกเขา!

หมายเหตุเกี่ยวกับการสนับสนุน DOS:

ถ้าคุณสร้างไดรฟ์ที่สามารถบูต DOS และใช้แป้นพิมพ์ที่ไม่ใช่ของสหรัฐอเมริกา Rufusจะพยายามที่จะเลือกรูปแบบแป้นพิมพ์ตามสถานที่จากระบบของคุณ หากเป็นเช่นนั้น FreeDOS ซึ่งเป็นตัวเลือกเริ่มต้นจะเหมาะสมกว่า MS-DOS เนื่องจากการสนับสนุนรูปแบบแป้นพิมพ์ที่หลากหลายกว่า

หมายเหตุเกี่ยวกับการสนับสนุน ISO:

ทุกรุ่นของRufus ตั้งแต่ v1.1.0 สามารถสร้างUSBที่บูทได้จาก ISO (.iso)

การสร้าง ISO จากแผ่นดิสก์ หรือจากไฟล์เป็นเรื่องง่ายมาก อย่างไรก็ตามคุณจะต้องใช้งานผ่านโปรแกรมเบิร์นCDที่มีสามารถดาวน์โหลดได้ฟรี เช่น InfraRecorder หรือ CDBurnerXP

การบริจาคเพื่อสนับสนุน

เนื่องจากมีการสอบถามเข้ามาเป็นจำนวนมาก ปุ่มบริจาคเงินจะไม่มี ในหน้านี้

เหตุผลหลักก็คือ ผมรู้สึกว่าระบบการบริจาคไม่ได้ช่วยในการพัฒนาซอฟท์แวร์ และที่แย่ไปกว่านั้นคือ ระบบบริจาคอาจจะทำให้ผู้ที่เลือกที่จะไม่บริจาครู้สึกผิดกับการที่ตัดสินใจแบบนั้น

แต่หากคุณยืนยันจริงๆว่าคุณต้องการจะบริจาค คุณสามารถทำการบริจาคได้ที่ Free Software Foundation เนื่องจากพวกเขาเป็นเหตุผลหลักที่ทำให้Rufusเกิดขึ้นได้

สุดท้ายนี้ ผมขอใช้โอกาสนี้กล่าวคำ ขอบคุณ สำหรับการสนับสนุนอย่างต่อเนื่องและล้นหลามในโปรแกรมเล็กๆโปรแกรมนี้ ผมรู้สึกซาบซึ้งจริงๆ

และโปรดใช้งาน Rufusได้อย่างเสรีและสบายใจแม้ว่าคุณจะไม่ได้สนับสนุนใดๆเลยทางด้านการเงิน คุณไม่ควรจะเลยที่จะรู้สึกแย่
