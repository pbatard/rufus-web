Rufus

간편하게 부팅 가능한 USB 드라이브 만들기

Rufus는 USB 메모리 및 플래시 드라이브를 포맷하고 부팅할 수 있도록 만드는 도구입니다.

이 프로그램은 다음 상황에서 유용하게 사용할 수 있습니다:

  • 부팅 가능한 ISO 파일(Windoows, 리눅스, UEFI 등)을 USB로 설치해야 할 때
  • OS가 설치되지 않은 시스템에서 작업해야 할 때
  • DOS 환경에서 BIOS나 기타 펌웨어를 설치해야 할 때
  • 로우포맷 유틸리티를 실행해야 할 때

Rufus의 크기는 작지만 필요한 모든 것이 들어 있습니다!

그리고 Rufus는 빠릅니다. ISO 이미지에서 Windows 7 설치 USB 드라이브를 만들 때 UNetbootin, Universal USB Installer, Windows USB download tool 보다 약 두 배 빠릅니다. 또, ISO 이미지에서 부팅 가능한 리눅스 USB 드라이브를 만들 때에도 더 빠르게 작동합니다. (1)
이 페이지 아래에 나열된 ISO 이미지 이외에도 Rufus는 여러 종류의 ISO 이미지를 지원합니다. (2)

시스템 요구 사항:

32비트/64비트 여부와는 무관하게 Windows 7 이상의 운영 체제에서 동작합니다. 다운로드만 하면 바로 사용할 수 있습니다.

Rufus 프로그램 및 이 웹 사이트를 여러 언어로 번역해 주신 모든 번역자분들께 감사의 인사를 드립니다. 사용하고 있는 언어로 Rufus가 번역되어 있다면 해당 번역자에게 감사의 말을 전해 주세요!

사용 방법

실행 파일을 다운로드하고 실행하기만 하면 됩니다. 프로그램을 설치할 필요가 없습니다.

실행 파일은 다음과 같이 전자 서명되어 있습니다:

  • "Akeo Consulting" (버전 1.3.0 이상)
  • "Pete Batard - Open Source Developer" (버전 1.2.0 이하)

DOS 지원 참고 사항:

DOS 부팅 USB 드라이브에서 미국 영어가 아닌 키보드를 사용해야 한다면, Rufus에서는 USB 드라이브를 만드는 시스템의 지역 설정에 따라서 키보드 레이아웃을 선택합니다. 미국 영어 이외의 키보드를 사용해야 하는 경우에는 MS-DOS 대신 FreeDOS(기본값) 사용을 권장합니다. FreeDOS에서는 더 많은 키보드 레이아웃을 지원하기 때문입니다.

ISO 지원 참고 사항:

Rufus 버전 1.1.0 이상에서는 ISO 이미지(.iso)로부터 부팅 가능한 USB 드라이브를 만들 수 있습니다.

무료로 사용 가능한 InfraRecorder, CDBurnerXP 등 CD/DVD 제작 프로그램을 사용해 CD/DVD에서 ISO 이미지를 생성하거나 필요한 파일이 들어 있는 ISO 이미지를 쉽게 생성할 수 있습니다.

자주 묻는 질문(FAQ)

Rufus에 대한 자주 묻는 질문은 여기(영어)를 참조하세요.

GPLv3 라이선스의 규정 하에 이 소프트웨어를 배포, 수정 및 판매할 수 있습니다.

Rufus는 공개된 소스 코드로부터 MinGW32 환경을 사용하여 100% 투명한 환경에서 빌드됩니다.

기부

기부 방법에 대해서 자주 문의를 받지만, 이 페이지에는 기부 버튼이 없습니다.

주된 이유는 기부 시스템은 실질적으로 소프트웨어 개발을 도와주지 않으며, 더 나쁘게 만든다고 생각하기 때문입니다. 또한 기부 시스템은 기부를 안 하는 사용자들이 죄책감을 지게 하기 때문입니다.

만약에 진정 기부를 원하시면, 자유 소프트웨어 재단에 기부를 해서 Rufus와 같은 소프트웨어가 만들어질 수 있도록 도와 주세요.

이 자리를 통해서 여러분이 이 작은 프로그램에 보여 주는 지속적인 지원과 관심에 감사의 말을 전하고 싶습니다.

Rufus 개발에 기여하지 않거나 금전적인 지원을 하지 않았다는 죄책감 없이 자유롭게 사용하시기를 바랍니다.
