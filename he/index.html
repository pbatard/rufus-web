Rufus

יצירת כונני USB ברי־אתחול בדרך הקלה

Rufus הוא כלי המסייע באתחול ויצירת כונני הבזק מסוג USB הניתנים לאתחול, כמו דיסק און קי, כרטיסי זיכרון וכו׳.

כלי זה יכול לסייע במקרים שבהם:

  • יש לך צורך ליצור מדיית USB של התקנה מקובצי ISO ברי־אתחול (Windows,‏ Linux,‏ UEFI וכו’)
  • יש לך צורך לעבוד על מחשב שלא מותקנת בו מערכת הפעלה
  • יש לך צורך לעדכן BIOS או קושחה אחרת מ־DOS
  • ברצונך להריץ כלי low-level

למרות גודלו הקטן, Rufus מספק את כל צרכיך!

אה, Rufus גם מהיר. למשל, המהירות שלו היא כפולה מיישומים כמו UNetbootin, ‏Universal USB Installer או Windows 7 USB download tool, ביצירת כונן USB להתקנת Windows 7 מתוך קובץ ISO. מהירותו גם גבוהה יחסית ביצירת כונן USB בר־אתחול של Linux מקובצי ISO‏. (1)
רשימה לא מקיפה של קובצי ISO הנתמכים ע״י Rufus נמצאת בתחתית העמוד הזה. (2)

דרישות מערכת:

מערכת הפעלה Windows 7 ומעלה, 32 או 64 סיביות. לאחר ההורדה, היישום יהיה מוכן לשימוש.

בהזדמנות זו אביע את תודתי למתרגמים שבזכותם Rufus, כמו גם דף האינטרנט הזה, מתורגמים לשפות שונות. אם אתה מגלה שניתן להשתמש ב־Rufus בשפה שלך, התודה באמת צריכה להגיע להם!

הערות על תמיכה ב־DOS:

אם תיצור כונן בר־אתחול מסוג DOS ותשתמש במקלדת שאינה מסוג US‏, Rufus ינסה לבחור בפריסת לוח מקשים אשר תואמת לאזור המוגדר במערכת שלך. במקרה זה, FreeDOS שהיא הבחירה ברירת המחדל, עדיפה מאשר MS-DOS, מכיוון שהיא תומכת ביותר פריסות לוח מקשים.

הערות על תמיכה ב־ISO:

כל הגרסאות של Rufus החל מגרסה 1.1.0 מאפשרות ליצור USB בר־אתחול מקובץ תמונת ISO (בסיומת iso).

יצירת קובץ תמונת ISO מדיסק פיזי או מקבוצת קבצים זה קל מאוד לביצוע, דרך שימוש ביישומים לצריבת CD כמו InfraRecorder ו־CDBurnerXP הזמינות להורדה בחינם.

שאלות נפוצות

ניתן לקרוא שאלות נפוצות על Rufus כאן (באנגלית).

רישיון

רישיון ציבורי כללי של GNU‏ (GPL) גרסה 3 ומעלה.
אתה רשאי להפיץ, לשנות או אפילו למכור את התוכנה, כל עוד אתה מכבד את תנאי הרישיון של GPLv3.

Rufus בנוי בצורה שקופה ב־100%, ממקורו הציבורי תוך שימוש בסביבת MinGW32.
