Rufus

Enostavno ustvarjajte zagonske pogone USB

Rufus je program za formatiranje in ustvarjanje zagonskih naprav USB.

Uporaben je še posebej, če:

  • morate ustvariti zagonski medij USB iz zagonskih slik ISO (Windows, Linux, UEFI, itd.)
  • morate delati na računalniku, ki nima nameščenega operacijskega sistema
  • morate uporabiti DOS, da posodobite BIOS ali drugo strojno programsko opremo
  • morate uporabiti nizkonivojsko orodje

Čeprav je majhen, Rufus ponuja vse, kar potrebujete!

In mimogrede - Rufus je hiter. Je npr. enkrat hitrejši kot UNetbootin, Universal USB Installer ali Windows 7 USB download tool pri ustvarjanju USB namestitve Windows 7 iz slike ISO. Je tudi malce hitrejši pri ustvarjanju zagonske naprave USB iz ISO slike Linuxa. (1)
Nepopoln seznam slik ISO, ki jih Rufus podpira, je priložen na dnu te strani. (2)

Sistemske zahteve:

Windows 7 ali novejši, vseeno, ali je 32- ali 64-bitni. Aplikacija je pripravljena za uporabo takoj po prenosu.

Lepo se zahvaljujem vsem prevajalcem, ki so prevedli Rufus in to spletno stran iz angleščine v različne jezike.

Opombe k podpori za DOS:

Če ustvarite zagonski medij za DOS in uporabljate neameriško tipkovnico, bo Rufus poskusil izbrati razpored tipk glede na tistega na vašem sistemu. V tem primeru je namesto MS-DOS priporočen FreeDOS (ki je privzeta možnost), ker podpira več razporedov.

Opombe k podpori ISO:

Rufus lahko od verzije 1.1.0 dalje ustvarja zagonske medije USB iz slike ISO (.iso).

Ustvarjanje slike ISO iz fizične plošče ali množice datotek je enostavno z uporabo aplikacije za peko CD-jev, kot je InfraRecorder ali CDBurnerXP.

Pogosto zastavljena vprašanja

Odgovori na pogosto zastavljena vprašanja o Rufusu so na voljo tukaj (v angleščini).

Če želite komentirati, prijaviti hrošča ali zahtevati izboljšavo (v angleščini), uporabite GitHubovo aplikacijo za sledenje težav, ali pa pošljite sporočilo po elektronski pošti.
