Rufus

Bootable USB ධාවක පහසුම ආකාරයෙන් සාදන්න

Rufus යනු USB ෆ්ලෑෂ් ඩ්‍රයිව් ආකෘතිකරණය කිරීමට සහ නිර්මාණය කිරීමට උපකාරී වන මෙවලමකි.

මේ අවස්ථා වලදී එය විශේෂයෙන් ප්‍රයෝජනවත් වේ:

  • boot කළ හැකි ISO වලින් (Window, Linux, UEFI ආදිය) ඔබ USB ස්ථාපන මාධ්‍යයක් නිර්මාණය කළ යුතුය.
  • මෙහෙයුම් පද්ධතියක් ස්ථාපනය කර නොමැති පද්ධතියක වැඩ කිරීමට ඔබට අවශ්‍යය
  • ඔබට DOS වෙතින් BIOS හෝ වෙනත් firmware ෆ්ලෑෂ් කිරීමට අවශ්‍යය
  • ඔබට පහත් මට්ටමේ උපයෝගීතාවයක් ක්‍රියාත්මක කිරීමට අවශ්‍යයි

කුඩා ප්‍රමාණය තිබියදීත්, Rufus ඔබට අවශ්‍ය සියල්ල සපයයි!

Rufus වේගවත්.උදාහරණයක් ලෙස එය UNetbootin, Universal USB Installer or Windows 7 USB download tool වඩා දෙගුණයක් වේගවත්, ISO වෙතින් Windows 7 USB ස්ථාපන ධාවකය නිර්මාණය කිරීම පිළිබඳ. ISO වෙතින් Linux boot හැකි USB නිර්මාණය කිරීමේදී ද එය සුළු වශයෙන් වේගවත් වේ. (1)
මෙම පිටුවේ යටRufus සහාය දක්වන ISO පරිපූර්ණ නොවන ලැයිස්තුවක් ද සපයනු ලැබේ. (2)

පරිත්යාග

මා මේ පිළිබඳව නිරන්තරයෙන් විමසන බැවින්, මෙම පිටුවේ නැත පරිත්යාග බොත්තමක් තිබේ.

ප්‍රධාන හේතුව නම්, පරිත්යාග පද්ධතිය ඇත්ත වශයෙන්ම මෘදුකාංග සංවර්ධනයට උදව් නොකරන බවත්, ඊටත් වඩා නරක බවත්, පරිත්යාග නොකිරීමට තීරණය කරන පරිශීලකයින්ට වරදකාරි හැඟීමක් ඇති කළ හැකි බවත් මට හැඟේ.

ඔබ සැබවින්ම අවධාරනය කරන්නේ නම්, ඔබට සැමවිටම නිදහස් මෘදුකාංග පදනම වෙත පරිත්‍යාගයක් කළ හැකිය, මන්ද රූෆස් වැනි මෘදුකාංග ලබා ගත හැකි ප්‍රධාන හේතුව ඒවා වන බැවිනි.
