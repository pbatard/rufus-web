Rufus

Método fácil de criar unidades USB inicializáveis

Rufus é um utilitário que ajuda a formatar e a criar unidades USB inicializáveis, tais como dispositivos USB/pendrives, cartões de memória, etc.

Pode ser especialmente útil nos casos em que:

  • precisa de criar uma unidade USB de instalação a partir de um ISO inicializável (Windows, Linux, UEFI, etc.)
  • precisa de trabalhar num computador que não tem um sistema operativo instalado
  • precisa de atualizar a BIOS ou outro firmware a partir do DOS
  • pretende executar um utilitário de baixo nível

Apesar de pequeno no tamanho, Rufus tem tudo o que precisa!

Ah, e o Rufus é rápido. É cerca de duas vezes mais rápido que o UNetbootin, o Universal USB Installer ou o Windows USB / DVD Download Tool, na criação de uma unidade USB de instalação do Windows 7 a partir de um ISO. É também ligeiramente mais rápido na criação de USB inicializável Linux a partir de ISOs. (1)
Uma breve lista de ISOs suportados por Rufus é fornecida na parte inferior desta página. (2)

Utilização

Transfira o executável e execute-o – não é necessário qualquer instalação.

O executável está assinado digitalmente e a assinatura deve indicar:

  • "Akeo Consulting" (v1.3.0 ou posterior)
  • "Pete Batard - Open Source Developer" (v1.2.0 ou anterior)

Notas sobre suporte DOS:

Se criar uma unidade inicializável de DOS e usar um teclado não US (EUA), o Rufus tentará selecionar um esquema de teclado de acordo com a localização do seu sistema. Neste caso, FreeDOS é a opção predefinida e recomendada em vez do MS-DOS, pelo facto de suportar mais configurações de teclado.

Notas sobre suporte ISO:

Todas as versões do Rufus, após v1.1.0, permitem a criação de um USB inicializável a partir de uma imagem ISO (.iso).

Criar uma imagem ISO de um disco físico ou a partir de um conjunto de ficheiros é um processo fácil de fazer, através da utilização de uma aplicação de gravação de CD, tais como o InfraRecorder ou o CDBurnerXP, ambas gratuitos.
